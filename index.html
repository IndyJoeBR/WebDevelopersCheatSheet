Basic Structure of a Web Page

Nearly every web page you create will require the following basic structure. (It assumes you are using CSS to style the web page.)

<!DOCTYPE html>
<html>
<head>
<title> /
<link> href="stylesheet URL" type="text/css" rel="stylesheet"
</head>
<body>
<header> /
<main> /
<footer> /
</body>
</html>

Basic Structure of a HTML tag

An HTML element is defined by an opening tag, content and a closing tag; although some elements - such as <br> - have no content or closing tag. In addition to content, an element may have one or more attributes. HTML attributes provide additional details about and to the tag.

HTML Attributes

Attribute Description Code Example
action Used with <form>, it provides the URL where collected data will be sent <form action="/example.html" method="POST">
alt Alternative text; provides a description typically used by assistive technology <img src="#" alt="A field of yellow sunflowers"/>
autoplay Used in audio/video tages; plays media when the page loads <audio autoplay>
border DEPRACATED; used within a <table> to set the border thickness <table border="1">
class Primarily used to point to a class in a style sheet <p class="brand"> Sole Shoe Company </p>
cols Used within <textarea>, it sets the number of coumns displayed <textarea id="blog" name="blog" rows="5" cols="30">
colspan Spanning columns: used within <th> or <td> elements, it spreads the content over the number of columns given as its value; also set the scope to "colgroup" <th rowspan="2" scope="rowgroup">Out of Stock</th>
controls Used with audio/video media, it instructs the browser to include basic controls: pause, play, skip <video src="myVideo.mp4" width="320" height="240" controls >
for Used within <label> and paired the id attribute of an <input>, it instructs the browser which field gets the label <XXXXXX> CODE EXAMPLE </XXXXX>
height Sets the height of a video display in the browser <video src="myVideo.mp4" width="320" height="240" controls>
href Hyperlink reference; provides the destination of a link <a href="https://www.wikipedia.org/"> This Is A Link To Wikipedia </a>
id An ID for an element, it describes different kinds of content <p id="intro">
list Used within <input> and paired the id attribute of a <datalist>, it allows the datalist to be used by the input field <input type="text" list="cities" id="city" name="city">
<datalist id="cities">
max Used within <input>, it sets a maximum value for the number and ranges types <input id="guests" name="guests" type="number" min="1" max="4">
maxlength Used within <input> of the "text" type, it sets a maximum number of characters allowed <input id="summary" name="summary" type="text" minlength="5" maxlength="250" required>
min Used within <input>, it sets a minimum value for the number and ranges types <input id="guests" name="guests" type="number" min="1" max="4">
minlength Used within <input> of the "text" type, it sets a minimum number of characters allowed <input id="summary" name="summary" type="text" minlength="5" maxlength="250" required>
name Used within <input>, essentially it is the variable name for the input <input type="text" name="first-text-field" value="already pre-filled">
onclick The action to be taken when an input button is clicked <button type="button" onclick="alert('Hello World!')" > Click Me! </button>
pattern Used within <input>, it defines the class of characters that can be entered <input id="payment" name="payment" type="text" required pattern="[0-9]{14,16}">
placeholder Used within <input>, its value appears in the field until the user begins to type <input type="text" name="search" class="form-control" placeholder="Search movies, TV, actors, more..." id="fullscreen-search-term" autocomplete="off" value="">
rel Relationship: specifies the relationship of a linked document to the current document (eg. such as the CSS document to index.html) <link rel="stylesheet" href="style.css">
required sed within <input>, it forces the user to enter data <input id="allergies" name="allergies" type="text" required#62
rows Used within <textarea>, it sets the number of rows displayed/allowed <textarea id="blog" name="blog" rows="5" cols="30">
rowspan Spanning rows: used within <th> or <td> elements, it spreads the content over the number of rows given as its value; also set the scope to "rowgroup" <th rowspan="2" scope="rowgroup">Out of Stock</th>
scope Used within <th> element, it defines the header as being for either a column ("col"), column group ("colgrou"), row ("row"), row group ("rowgrou") or auto (default value) <th scope="col">Saturday</th>
src Source: used similar to 'href', it points to the source of a file, either a url or a local address; it is required for image and video tags <img src="image-location.jpg">
step Used within <input> field of the "number" type, it adds +/- arrows to field and allows a value by which they will increase/decrease the entered value <input id="years" name="years" type="number" step="1">
style Useable with most opening tags to add CSS modifications directly in the web page <p style="color: red;"> I'm learning to code! </p>
target Used within <input>, it sets how to open the link (_blank, _self, _parent, _top) <a href="https://en.wikipedia.org/wiki/Brown_bear" target="_blank">
type Used within <form>, it defines the type of field the browser will open; options are: checkbox, button, numbers, password, radio, range, submit, text <input type="text" name="first-text-field" value="already pre-filled">
value Used within <input>, it is typically paired with the name attribute; when the user enters a value, it is held by this attribute; optionally it can hold a default value <input type="text" name="first-text-field" value="already pre-filled">
width Sets the width of a video display in the browser <video src="myVideo.mp4" width="320" height="240" controlsX>
